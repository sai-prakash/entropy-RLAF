# Security Policy

## Supported versions

This repository is pre-1.0 and receives best-effort security updates on the latest main branch.

## Reporting a vulnerability

Please disclose vulnerabilities responsibly by opening a private security advisory or by contacting maintainers directly.

Include:
- reproduction steps,
- impact analysis,
- proof-of-concept payload,
- suggested mitigation.

Do not publish zero-day details before a fix is available.
